# MacParakeet

**The fastest local transcription app for Mac.**

MacParakeet uses NVIDIA's Parakeet TDT model to transcribe audio at up to **300x realtime** - entirely on your Mac, with zero cloud uploads.

## Features

- 🚀 **Blazing Fast** - Transcribe a 3-hour podcast in under 2 minutes
- 🔒 **100% Private** - Audio never leaves your Mac
- 💰 **One-Time Purchase** - No subscriptions, no accounts
- 🎯 **Simple** - Drag, drop, done

## Why MacParakeet?

Unlike cloud transcription services, MacParakeet:
- Runs completely offline
- Requires no internet connection
- Never uploads your audio anywhere
- Works with Apple Silicon (M1/M2/M3/M4)

## Requirements

- macOS 14.2+ (Sonoma)
- Apple Silicon Mac (M1 or later)
- 8GB RAM minimum (16GB recommended)

## Installation

Download from [macparakeet.com](https://macparakeet.com)

## Usage

### Transcribe a File
1. Drag audio/video file onto MacParakeet
2. Wait for transcription
3. Copy or export result

### System-Wide Dictation
1. Press `⌥⌥` (double-tap Option)
2. Speak
3. Release to paste text

### Supported Formats
- Audio: MP3, WAV, M4A, FLAC, OGG
- Video: MP4, MOV, MKV, WebM

### Export Formats
- Plain Text (.txt)
- Subtitles (.srt, .vtt)
- Word Document (.docx)

## Privacy

MacParakeet processes everything locally using the Parakeet TDT model. Your audio is never:
- Uploaded to any server
- Stored outside your Mac
- Used for training AI models

## Tech Stack

- **STT Engine**: Parakeet TDT 0.6B-v3 via MLX
- **Framework**: Swift + SwiftUI
- **Platform**: macOS 14.2+ (Apple Silicon)
